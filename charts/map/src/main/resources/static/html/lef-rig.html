  • 点层图方案

    简介
    灵活多变的点状控制图层的显示状态，将数据完美地展示出来，使地图更美观
    使用
    平台还提供地图控件、地图基础组件、地图高级组件三大类组件供使用，可根据需要将组件添加到展示区域，辅助可视化展示平台生成的可视化项目， 可导出为图片放入文档中，也可以通过URL直接在浏览器中访问或嵌入到网站中

  • 自定义图层解决方案

    简介
    自己定制区域面、建筑物、水系、天空、道路、标注、行政边界等地图元素,随心所欲的设计心仪的地图样式
    使用
    地图元素可供设置是否显示、填充颜色、描边颜色、不透明度等多种属性，根据场景选择性显示地图元素，地图更简洁，更美观
